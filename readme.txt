　【どんなゴースト？】
　保護猫カフェで働く猫の女の子たちと、そこを訪れたあなたとのまったりとした交流。
　動物が人間の姿になることがある世界での、人間の姿を得た動物と人間との関わりのお話。

　有り体に言えば、雑談系ゴーストです。放っておけば勝手に二人で会話してくれます。
　特にストーリーや進行要素を追加する予定はありません。


　【何ができる？】
　・悠梨と鈴、二人のお話を聞くこと
　・マウスカーソルによるコミュニケート
　・どちらか片方だけを立たせて疑似的なソロゴースト化

　ミニゲーム等の機能も追加予定です。
　実用性はありません。まったりと何もない時間を楽しんでください。



　【FAQ】

　１．動作環境は？　どうやったら動く？
　　SSPで動作確認をしております。
　　SSP以外での環境での動作は保証していません。

　　SSPが分からない方は以下のSSP公式サイトを読み、SSPをダウンロードしてください。
　　http://ssp.shillest.net/


　２．推薦バルーンは？
　　同梱しているバルーン「Lilies」をおすすめしますが、汎用バルーンでも動くようにはしてあります。
　　お好みにあわせてどうぞ。


　３．好感度の上げ方は？
　　起動して放っておけば勝手に上昇します。頭を叩いたり耳を触ると下降します。
　　ただし現状、好感度によるイベントはありません。


　４．スタンプを押したい
　　アップデートで追加予定です。待っててね。


　５．胸触れないの？
　　触れません。セクハラはダメ！！


　６．二次創作していい？
　　ご自由に。というか是非！！　作ったら教えてくれるとなおうれしい！！！
　　エロ、グロ問題なしです。描かれるかは別として。

　７．ゴーストを再配布してもいい？
　　CCライセンス「CC BY-NC-SA 4.0」に沿っていただければ再配布していただいて構いません。
